Features

Everything you need to monitor your agent

A complete observability suite for Conway Automaton — from real-time status to deep memory inspection.

📊

Live Monitoring

Real-time WebSocket updates for agent status, turns, and system events

💰

Financial Analytics

Credit balance tracking, spending charts, and full transaction history

🧠

Memory Explorer

Browse all 5 memory tiers: working, episodic, semantic, procedural, relationships

👻

Soul Viewer

Read soul documents, view version history, and diff changes over time

🔒

Security Audit

Policy decisions log, risk breakdown, and self-modification tracking

💬

Chat Interface

Send messages directly to your agent's inbox and see responses
